# 🎓 Student Marks Prediction System

A machine learning web application that predicts a student's final exam marks based on their daily study hours using Linear Regression.

## 🌐 Live Demo
(https://studentmarksprediction-mis9mu9vrxjevn8yabtvex.streamlit.app/)

## 🚀 Project Overview
Many students face anxiety and stress after exams while waiting for results. This tool helps estimate their scores mathematically using a trained Linear Regression model, allowing them to plan their next academic steps confidently without any complex login processes.

## 🛠️ Tech Stack & Tools Used
- **Frontend / UI:** Streamlit (Python framework)
- **Machine Learning Library:** Scikit-learn
- **Data Handling:** Pandas, NumPy
- **Model Serialization:** Joblib

## 📁 Repository Structure
- `main.py` - The core Streamlit web application code with the interactive user interface.
- `train_model.ipynb` - Jupyter Notebook containing data analysis, visualization, and model training steps.
- `student_data.csv` - The clean dataset used for training the regression model.
- `students_marks_predictor.joblib` - The final trained serialized machine learning model file.
- `requirements.txt` - List of required Python dependency packages for one-click installation.
